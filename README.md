## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing.

1. **Clone the repository:**

   ```bash

   cd boostore-app/

2. **Installed Dependencies:**

   ```bash
   npm install

2. **Run Project:**

   ```bash
   npm start
